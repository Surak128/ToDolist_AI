# ToDolist_AI

โปรแกรม To-Do List (CLI) แบบง่ายเขียนด้วย Python

## คุณสมบัติหลัก
- เพิ่มงาน (title, description, due_date)
- ดูงานทั้งหมด (แสดงลำดับ, ชื่อ, วันครบกำหนด, สถานะ)
- แก้ไขงาน (เลือกโดยลำดับ/index)
- ลบงาน (เลือกโดยลำดับ/index พร้อมยืนยัน)
- บันทึก/โหลดงานอัตโนมัติจาก `tasks.json`

## ติดตั้ง
ต้องการ Python 3.x (แนะนำ 3.8+)

ไม่มี dependency ภายนอก — ใช้งานได้ทันที

## การรันโปรแกรม
จากโฟลเดอร์โปรเจค ให้รัน:

```bash
python main.py
```

เมื่อเริ่มโปรแกรม จะโหลดงานจากไฟล์ `tasks.json` (ถ้ามี) และเมื่อออกจากโปรแกรมจะบันทึกกลับไปให้โดยอัตโนมัติ

## เมนูตัวอย่าง (การใช้งาน)
1. เพิ่มงานใหม่
   - เลือก `1` แล้วกรอก `ชื่อเรื่อง`, `รายละเอียด`, `วันครบกำหนด` (รูปแบบ YYYY-MM-DD)
2. ดูงานทั้งหมด
   - เลือก `2` โปรแกรมจะแสดงรายการงานพร้อมสถานะ
3. แก้ไขงาน
   - เลือก `3` แล้วกรอกลำดับ (index) ของงานที่ต้องการแก้ไข
   - กด Enter หากไม่ต้องการเปลี่ยนช่องใดช่องหนึ่ง
   - ป้อน `y` หรือ `n` เพื่อเปลี่ยนสถานะเสร็จ
4. ลบงาน
   - เลือก `4` แล้วกรอกลำดับ (index) ของงานที่ต้องการลบ
   - ยืนยันด้วย `y` เพื่อทำการลบ
5. ออกจากโปรแกรม
   - เลือก `5` โปรแกรมจะบันทึก `tasks.json` แล้วปิด

## ไฟล์ข้อมูล
- `tasks.json` — เก็บรายการงานในรูปแบบ JSON (ถูกสร้างอัตโนมัติเมื่อบันทึก)

## ตัวอย่างการทดสอบแบบไม่โต้ตอบ (Windows cmd)
```cmd
(echo 1&echo งานตัวอย่าง&echo รายละเอียด&echo 2026-05-01&echo 5) | python main.py
```

## ถ้าต้องการเพิ่มเติม
- บันทึกทุกครั้งที่มีการเปลี่ยนแปลง
- เพิ่มการค้นหา/กรองงานตามสถานะหรือวันครบกำหนด
- เพิ่มการทำงานแบบ GUI หรือเว็บ
